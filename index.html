Setup Repository Termux (Gratis)

Salin dan tempel (Paste) perintah di bawah ini ke terminal **Termux** Anda untuk menginstal semua dependensi.

$ pkg update -y && pkg upgrade -y
$ pkg install git nodejs termux-tools python -y
$ git clone https://github.com/GMhabib/nextjs.git
$ cd nextjs
$ npm run build
$ npm start

Catatan: Pastikan Anda menjalankan `node server.js` di direktori yang benar (`vps-webshell`) setelah instalasi.
