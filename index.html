Setup Repository Termux (Gratis)

Salin dan tempel (Paste) perintah di bawah ini ke terminal Termux Anda untuk menginstal semua dependensi yang diperlukan.

$ pkg update -y && pkg upgrade -y
$ pkg install git nodejs termux-tools python -y
$ git clone https://github.com/[USERNAME-ANDA]/vps-webshell
$ cd vps-webshell
$ npm install
$ node server.js

Catatan: Pastikan Anda menjalankan `node server.js` di direktori yang benar setelah instalasi.
